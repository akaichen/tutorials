WEBDUINO

WEB × ARDUINO

新一代的創客體驗

使用 Webduino 將可以讓不具有專業背景的設計師、網頁開發者，也可以輕鬆成為創客 ( Maker、自造者 )，享受創造的樂趣和快感。

純粹使用網頁語法開發

Webduino 純粹利用 HTML、Javascript ...等網頁語法進行開發，不需要學習艱難的 C 或 C++，只要會寫網頁，就可以串起 Arduino，控制形形色色的傳感器。

任何平台均可運行

由於使用網頁語法開發，可以藉由網頁控制元件或傳感器，因此不論是 Windows、Mac、Android 或 iOS 都可以順利運行，輕鬆實現跨平台的控制。

輕鬆打造物聯網應用

Webduino 的誕生，替物聯網開啟了最關鍵的一條路，讓網頁開發者可以更容易地跨入硬體的領域，讓硬體的開發者可以更簡單的使用網頁的服務。

網頁與傳感器的化學反應

形形色色的傳感器，搭配取之不盡的 Web API 和 Web Components，塑造了千變萬化的物聯網世界，許許多多的創意正在醞釀發酵著。

快速上手的開發模式

藉由教學範例手把手地進行，由淺入深的一步步學習，在很短的時間內就能掌握 Webduino 的開發節奏，創作自己獨一無二的應用。
