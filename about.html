關於我們

我們是一個高雄的熱血創意團隊，在 2015 年，實現了真正用 Web 串接 Arduino 的完整過程，純粹撰寫 HTML 與 Javascript 的網頁語法，就能夠玩轉各式各樣的傳感器，為廣大的開發者和網頁設計師，創造了前所未有的開發模式。

我們目前進駐在「高雄軟體科學園區」，因此許多的課程和活動都會先從高雄舉辦
